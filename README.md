# My STARBUCKS Kiosk
C로 만든 TOY 프로젝트 


## 1. 개발 기간 및 참여 인원

+ 2023.07.21. ~

+ 개인 프로젝트

## 2. 기술 스택 및 개발 환경

+ Ubuntu 22.04.2 LTS
+ C (컴파일러 gcc 11.3.0)
+ NCURSES 
    ```
    sudo apt-get install libncurses5-dev libncursesw5-dev 
    ```

## 3. 프로그램 실행 방법

```
make clean 
make all
./server 포트번호
./client ip주소 포트번호
```
